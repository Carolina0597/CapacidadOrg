# Ajustes correos, formulario y asistencia

- El buzón abre por defecto el catálogo completo de correos simulados y cada ejemplo se puede abrir.
- Se muestran todos los ejemplos aun cuando exista un estado antiguo en localStorage.
- El formulario conserva únicamente Cancelar, Guardar borrador, Anterior, Siguiente y Enviar a For+.
- La programación muestra confirmación de disponibilidad cuando no existen cruces y alertas detalladas cuando sí existen.
- Al enviar el evento se genera confirmación para el área promotora y aviso para For+.
- La aprobación opcional de invitados definidos no se envía al crear o enviar el evento: se activa después de la aprobación de For+.
- El evento de demostración pendiente de asistencia incluye asistentes precargados.
- Se incrementó la versión del estado local para cargar los nuevos datos de demostración.
