# Ajustes de roles, acceso y Home For+

- El selector de rol muestra siempre los cuatro perfiles: Colaborador, Líder, Área promotora y For+ / Comité.
- Al seleccionar un rol que el usuario actual no posee, la app cambia automáticamente al usuario de demostración correspondiente.
- El rol Líder abre con Alejandro Gómez y su Home específico.
- El colaborador tiene visible en el menú la opción “Solicitar acceso para publicar”.
- El Home For+ fue simplificado para evitar métricas y acciones repetidas.
- El nuevo Home prioriza decisiones pendientes, accesos, ANS, cierres, capacidad, hitos y alertas accionables.
